# ush-0.0.3 不満な部分
- コマンド名のtab補完はできるが、ファイル名の補完が仕様外
- コマンド区切り `;`
- バックグラウンド実行 `&`
- グルーピング `(...)` / `{ ...; }`
- `if/then/elif/else/fi`
- `for/while/until/do/done`
- `case/esac`
- 関数定義
- バックスラッシュエスケープ（入力行の `\`）
- `${VAR}`
- 位置パラメータ（`$1`, `$@`, `$#` 等）
- コマンド置換 `$(...)` / `` `...` ``
- 算術展開 `$((...))`
- グロブ `* ? [ ]`
- 多段パイプ
- stderr リダイレクト（`2>` / `2>&1` 等）
- `<<`（ヒアドキュメント）、`<<<`（ヒアストリング）
- FD 操作（`n>file` など）
