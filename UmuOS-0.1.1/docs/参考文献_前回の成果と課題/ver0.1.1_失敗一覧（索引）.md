# ver0.1.1 失敗一覧（索引）

目的：後続PJで“同じ症状が出たときに”最短で当たりを付ける。

本索引は、詳細は [ver0.1.1_失敗の振り返り.md](ver0.1.1_%E5%A4%B1%E6%95%97%E3%81%AE%E6%8C%AF%E3%82%8A%E8%BF%94%E3%82%8A.md) に集約し、ここでは症状と参照先を一覧化する。

---

## ブート/FS

- F-01 initrd に `/init` が無いように見える
  - 典型症状: `lsinitramfs | head` に出ない
  - 参照: 実装ノート 2.2.1

- F-02 `mount: マウント済みです` が出て混乱する
  - 典型症状: mount が exit=1 に見える
  - 参照: 実装ノート 2.2.2

- F-03 `rcS` で止まる / getty が出ない
  - 典型症状: `login:` が出ず、ログも進まない
  - 原因候補: BusyBox applet symlink が壊れている
  - 参照: 実装ノート 2.2.3.1

- F-04 ext4 checksum エラー / fsck が必要
  - 典型症状: e2fsck がエラー、以後の変更が信用できない
  - 参照: 失敗の振り返り 5章（F-04）

## 認証/設定ファイル

- F-07 root/tama がログインできない
  - 典型症状: `login:` は出るが認証が通らない
  - 原因候補: `/etc/shadow` がプレースホルダのまま、権限不正
  - 参照: 実装ノート 2.2.5

- F-08 作ったはずのファイルが無い／権限が違う
  - 典型症状: `/etc/shadow` や `/etc/init.d/rcS` が存在しない
  - 原因候補: 途中中断（^C）・確認不足
  - 参照: 実装ノート 2.2.4（7.3〜7.5）

## ネットワーク

- F-05 Proxmox `vmbr0` 不調（ハング疑い）
  - 典型症状: LAN疎通/管理系が不安定、影響がVM/ホストに波及
  - 参照: 実装ノート 2.2.4、詳細設計 3.3.1

- F-06 telnet 接続が即切断
  - 典型症状: hostfwd で LISTEN していても接続が切れる
  - 原因候補: telnetd起動はしているがlogin/shadowが未確定
  - 参照: 失敗の振り返り 4.5〜4.6
