# uim-0.0.2

uim は UmuOS 向けの最小 vim ライクエディタです。

実装の範囲は「最小の操作を確実に動かす」ことに絞っています（未実装の vi 機能が多数あります）。

## 起動

```sh
cd uim
./uim <file>
```

既存ファイルが無い場合は空バッファとして起動します。

## ビルド/スモーク

`uim/tests/smoke_uim.sh` が `musl-gcc -static` でビルドし、`--batch` でスモークを実行します。

スモークには、日本語（UTF-8）入力だけでなく、日本語パターンの `/` 検索も含まれます。

```sh
cd uim
bash tests/smoke_uim.sh
```

## モード

- NORMAL: 移動・削除/ヤンク/貼り付け・検索・コマンド
- INSERT: テキスト入力
- COLON: `:` から始まるコマンド入力（保存/終了）
- SEARCH: `/` から始まる検索入力

## 使い方（キー操作一覧）

### NORMAL モード

移動:
- `h` / `j` / `k` / `l` : 左/下/上/右
- `↑` `↓` `←` `→` : 上記と同等
- `0` : 行頭
- `$` : 行末
- `Home` / `End` : 行頭/行末
- `gg` : 先頭行へ
- `G` : 最終行へ

編集:
- `i` : INSERT へ
- `a` : カーソルを 1 文字右へ動かして INSERT へ

行操作（vi 風の 2 打鍵コマンド）:
- `dd` : 現在行を削除（削除した行はヤンクにも入ります）
- `yy` : 現在行をヤンク
- `p` : ヤンクした行（複数行可）を「現在行の下」に貼り付け

検索:
- `/pattern` + Enter : 前方検索（リテラル一致、大小区別あり）
- `n` : 直前の検索を前方に繰り返し

補足:
- UTF-8（日本語）も検索できます（バイト列として一致）。

コマンド:
- `:` : COLON へ
- `Ctrl-C` : 終了

数値プレフィクス（例: `3dd`）:
- `Ndd` : N 行削除（削除した行はヤンクにも入ります）
- `Nyy` : N 行ヤンク
- `NG` : N 行目へ（例: `10G`）
- `Ngg` : N 行目へ（例: `10gg`）

### INSERT モード

- 通常入力 : 文字を挿入
- `Enter` : 改行（行分割）
- `Backspace` : 直前の 1 文字削除（UTF-8 を壊さない範囲で削除）
- `↑` `↓` `←` `→` / `Home` / `End` : INSERT のまま移動
- `ESC` : NORMAL に戻る

### COLON モード（`:`）

入力中:
- `Backspace` : 1 文字削除
- `ESC` : キャンセルして NORMAL へ
- `Enter` : 実行

コマンド:
- `:w` : 保存
- `:q` : 終了（未保存変更がある場合は拒否）
- `:q!` : 破棄して終了
- `:wq` : 保存して終了

### SEARCH モード（`/`）

入力中:
- `/pattern` と入力して `Enter` で検索実行
- `Backspace` : 1 文字削除
- `ESC` : キャンセルして NORMAL へ

挙動:
- 検索開始位置は「カーソルの次の文字」
- バッファ終端まで（ラップアラウンド無し）
- 見つからない場合 `Pattern not found`

## `--batch` について（スモーク用）

`--batch` を付けると、標準入力から疑似キー入力を読み取り、画面描画なしに編集します。

```sh
cd uim
printf 'iHello\033:wq\n' | ./uim --batch /tmp/out.txt
```

- `\033` は ESC
- `\n` は Enter として扱われます

## 制限（未実装）

uim-0.0.2 は最小機能のため、例えば以下は未実装です。

- 逆方向検索、ラップアラウンド検索、正規表現検索
- undo/redo
- 置換、範囲選択（visual）、複数ファイル編集

描画は差分描画ではなく、可視領域全体の上書き再描画です（チラつき低減のため、毎フレームの clear はしません）。
